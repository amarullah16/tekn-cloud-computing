# Konfigurasi Git
Gunakan perintah berikut:
```
$ git config --global user.name "amarullah"
$ git config --global user.email "amarullahahmad16@gmail.com"
```
Untuk melihat konfigurasi yang sudah ada:
```
$ git config --list
user.email=amarullahahmad16@gmail.com
user.name=amarullah
color.ui=true
$
```
